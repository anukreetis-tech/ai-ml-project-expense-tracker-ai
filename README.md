## EXPENSE TRACKER

This is a simple AI-based expense tracking application built in Python. It helps students manage their spending by automatically categorizing expenses, tracking budget usage, and giving real-time alerts when limits are reached or exceeded.

---

### FEATURES

AUTOMATIC CATEGORIZATION: Uses Machine Learning to classify expenses into categories like Food, Travel, Entertainment, Stationary, Fashion, and Essentials.

SELF-LEARNING SYSTEM: If the prediction is incorrect, the user can correct it and the model learns from that input.

BUDGET TRACKING: Set a monthly budget and track your spending.

REAL-TIME ALERTS:

* Warning when 80% of budget is used
* Notification when budget is reached
* Alert when budget is exceeded

DATA VISUALIZATION: Displays category-wise spending using bar graphs.

SPENDING PREDICTION: Predicts future expenses using a simple regression model.

GRAPHICAL USER INTERFACE: Easy-to-use interface built with Tkinter.

---

### PREREQUISITES

You need Python installed on your system.

Python 3.6 or higher is recommended.

Install required libraries:

```bash
pip install pandas matplotlib scikit-learn numpy
```

---

### HOW TO RUN THE PROGRAM

1. Save the code in a file named:

```bash
expense_tracker.py
```

2. Open terminal or command prompt.

3. Navigate to the folder where the file is saved.

4. Run the program:

```bash
python expense_tracker.py
```

---

### HOW TO USE

SET BUDGET:

* Enter your budget in the input field
* Click “Set Budget”

ADD EXPENSE:

* Enter description (e.g., pizza, uber, shirt)
* Enter amount
* Click “Add Expense”

VIEW SUMMARY:

* Click “Show Summary” to see total and category-wise spending

CHECK BUDGET STATUS:

* Click “Budget Status” to see remaining budget

PREDICT SPENDING:

* Click “Predict Spending” to estimate next expense

---

### IMPORTANT NOTES

* Always run it as a `.py` file
* Keep `expenses.csv` in the same folder as the script
